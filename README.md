# NEFramework

###### NEFramework由以下模块构建而成：
1. nbase基础框架，弱回调、Task机制、线程管理等
2. 优化过的duilib，控件class、控件自适应属性、窗口管理等
3. cef集成和封装为duilib控件
4. net模块： libcurl封装
5. 其它客户端模块


##### 目录:
1. bin : 演示demo_d.exe
2. libs :第三方库和框架基础库
3. toolkits/base : 网易客户端基础框架
4. toolkits/duilib : 经优化过的duilib
5. toolkits/net : libcurl http封装 
6. shared: cef的duilib control 封装、日志等
